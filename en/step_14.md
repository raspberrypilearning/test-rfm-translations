<h2 class="c-project-heading--task">6A - Draw Exit Sprite</h2>

Draw an `Exit` sprite that shows where the player should finish the level.

### Choose this route if...

You want a door, portal, flag, treasure, or finish marker that matches your own backdrop.

### Build it

Choose **Paint** in the sprite menu. Draw a clear finish object and name the sprite `Exit`.

Type your own exit position into the white inputs if you want it to start in a fixed place.

Add this code to the Exit sprite.

```blocks3
when green flag clicked
show
go to x: () y: ()
```

<h2 class="c-project-heading--task">Test</h2>

Click the green flag and check that the `Exit` sprite appears where the player can reach it.
