# Pathway Map

This maintainer note records the branching structure for **Branching Pathways Platformer**.

The learner-facing project contains exactly 31 alphanumeric route steps. Group titles describe the PDF structure but are not separate learner steps.

## Groups

- **1. Add a backdrop**: `1A - Draw Backdrop`, `1B - Upload Backdrop`, `1C - Choose Inbuilt Backdrop`
- **2. Add a player**: `2A - Draw Player`, `2B - Upload Player`, `2C - Choose Inbuilt Player`
- **3. Add platforms**: `3A - Add platforms to match your backdrop`, `3B - Draw foreground platforms`, `3C - Draw Platforms in another editor`
- **4. Move player**: `4A - Keys`, `4B - Mouse Move`, `4C - Always Moving`
- **5. Player jump/fall**: `5A - Player Jump-Fall (Keys)`, `5B - Player Jump-Fall (Mouse)`
- **6. Exit sprite**: `6A - Draw Exit Sprite`, `6B - Upload Exit Sprite`, `6C - Choose Exit Sprite from Library`
- **7. Add collectables**: `7A - Place Collectables`, `7B - Random Collectables`, `7C - Add a Power Up`
- **8. Win condition options**: `8A - Get to Exit Sprite (Default)`, `8B - Target Score`, `8C - Take Key to Exit`
- **9. Hazards**: `9A - Static Hazard`, `9B - Y-motion Hazards`, `9C - X-motion Hazards`
- **10. Extras**: `10A - Add Music`, `10B - Add Sound effects`, `10C - New Level`, `10D - Animate your player`, `10E - Make Your Player Sprint`
